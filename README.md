The system is designed to facilitate the entire election process, from candidate and voter accreditation to the final declaration of the winner. Through a comprehensive architecture, data flows seamlessly from the accreditation phase to the voting process, where votes are streamed to Kafka and processed with PySpark for real-time analysis. The project culminates in a dynamic dashboard built with Streamlit, providing real-time insights into voting trends and results. This endeavor represents an innovative application of data engineering principles in the realm of election technology, showcasing the integration of various technologies to create a robust and efficient voting system.
